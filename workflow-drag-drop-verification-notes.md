# Workflow Studio Drag-and-Drop Verification Notes

تم فتح صفحة `/workflow-studio` في بيئة المعاينة بعد إضافة واجهة السحب والإفلات. أظهرت الصفحة قسم **Interactive phase route** مع النص الإرشادي: "Drag phases to reorder the route, then save to persist the sequence with owner roles, permissions, gates, and evidence rules." كما ظهرت بطاقات المراحل بعناصر تفاعلية قابلة للسحب، وكل بطاقة تعرض إرشاد **Drag to reorder route** مع المرحلة والدور والصلاحية وحالة RBAC.

نتيجة الفحص البصري: خرجت الصفحة من حالة التحميل، واستدعت بيانات Workflow من الواجهة المتصلة بالـ API، وتعرض تجربة إعادة ترتيب مرئية مع استمرار أزرار التحريك البديلة في محرر المرحلة.
