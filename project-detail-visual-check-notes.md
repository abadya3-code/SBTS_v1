# Project Detail Visual Check Notes

تم فتح صفحة `/projects` في المعاينة، وظهرت كروت المشاريع الثلاثة مع أزرار `Open project` كرابط فعلي لكل مشروع. عند النقر على أول مشروع انتقل النظام إلى `/projects/PRJ-1033` بنجاح.

في صفحة تفاصيل المشروع ظهرت العناصر التالية بشكل صحيح:

- عنوان المشروع `North Manifold Isolation` مع رقم المشروع والمنطقة المرتبطة.
- بطاقات مؤشرات تشمل `Planned blinds` و`Project progress` و`Priority exposure` و`Inspection ready`.
- موجز تنفيذي للمشروع مع إجراء تشغيلي مقترح.
- توزيع مراحل الستائر: Broken / Preparation، Assembly، Tight & Torque، Final Tight، Inspection Ready.
- جدول `Detailed blinds for this project` يعرض سجلات الستائر المرتبطة بالمشروع، ويشمل tag، type/size، phase، priority، owner، isolation point، notes.
- الشريط الجانبي يظل على `Projects` كعنصر نشط أثناء صفحة التفاصيل، مما يحافظ على السياق الملاحي.

النتيجة: الربط البصري والتنقلي بين المشاريع وتفاصيل سجلات الستائر يعمل كما هو متوقع في المعاينة.
